Este proyecto fue realizado para una asociación sin fines de lucro, Asociación Patitas, con el fin de hacer una pagina web informativa con la cual la asociación pueda hacerse más conocida. Es totalmente responsive, cuenta con un hosting gratuito de Heroku y fue realizada con las tecnologias de HTML5 (Bootstrap), CSS3, ES5 (Jquery)
